BlackJack

O projeto consiste em fazer uma cópia simples do jogo BlackJack/21. Ao acessar a página há a interação se o usuário gostaria de comprar uma nova carta, se for negativo, o jogo finaliza, mas se for afirmativo, ele irá comprar uma nova carta, se chegar a 21 pontos ou passar, o jogo para, mas se for menos que 21, o programa pergunta se o usuário gostaria de comprar outra carta.
